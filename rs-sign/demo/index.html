④ rs-pdf 連携 — 署名/印影をPDFページに配置し注釈として書き出し

rs-sign の署名/印影（画像＋%座標＋ページ）を createPdfSignAdapter(pdfViewer) で rs-pdf の 画像スタンプ注釈に変換し、実際のPDFページ上へ配置します（rs-pdf のソースは変更せず公開契約 annotations.add を呼ぶだけ）。 ここではデモ用の合成ページを rs-pdf に読ませ、認印を押して annotations.toJSON() で書き出します。

⑤ rs-form 連携 — signature 質問に rs-sign 署名パッドを差し込み

createRSForm(el, schema, { signaturePad: rsSignaturePad({...}) }) の1行で、rs-form の signature 質問の 既定 canvas を rs-sign の滑らかな署名パッドに置き換えます（rs-form のソースは変更せず公開アダプタ契約に沿うだけ）。 描くと回答JSONに PNG dataURL が載ります。
