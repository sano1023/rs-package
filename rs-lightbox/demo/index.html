rs-lightbox v0.1 デモ — 依存ゼロの画像ライトボックス

サムネイルをクリックで開きます。矢印キー / Home / End で移動、Esc で閉じる、 ダブルクリック（ダブルタップ）や Ctrl+ホイール でズーム、ズーム中はドラッグでパン。 スマホは横スワイプで前後、縦スワイプで閉じる、ピンチでズームできます。

1. DOM 宣言モード（data-rslb-item をイベント委譲で開く）

<a href="大画像" data-rslb-item data-caption="..."><img src="サムネ" alt="..."></a> の形。修飾キーなしの通常クリックでのみ既定遷移を抑止します（Ctrl+クリックは新規タブ）。

2. 配列モード（ギャラリー DOM 無しで命令的に開く）
