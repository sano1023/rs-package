イベントログ

左のパレットから質問をキャンバスへドラッグ＆ドロップ（クリックでも追加）。カードは並べ替え・削除でき、選択すると右のパネルでラベル・選択肢・必須・バリデーション・表示条件（visibleIf）を編集できます。下部でライブプレビュー（標準／対話の切替）とスキーマJSONの入出力を確認でき、上部の元に戻す／やり直すで undo/redo できます。作ったスキーマは createRSForm() にそのまま渡せる素のJSONです。

複数の回答（回答JSONの配列）を質問ごとに集計し、rs-chart で可視化します（選択肢の円グラフ・NPS分布・星評価の分布・数値の要約統計）。rs-chart は読み取り連携（import して使うだけ）で、未ロード時は自動で表フォールバックに切り替わります。下のボタンで切り替えて確認できます。

CSSカスタムプロパティ（--rsf-*）を GUI で編集し、右のプレビューに即時反映します。色や角丸を変えると --rsf-primary などがプレビュー要素に適用されます。exportCSS() で :root に貼れるCSSも書き出せます。

多言語スキーマ: ラベル等を { ja, en } 形式で持てます（単一文字列は完全後方互換）。言語ボタンで form.setLocale() を呼ぶと回答を保ったまま表示言語が切り替わります。A/Bバリアント: variantSeed（利用者ID等）から決定的にバリアントを割り当て、variant タグの質問を振り分け・variants 上書きで文言を差し替えます。埋め込みスニペット: 現在の言語／バリアントで、任意サイトに貼れる HTML/JS を書き出します。
