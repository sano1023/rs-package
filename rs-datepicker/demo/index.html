rs-datepicker v0.1 デモ — 日本語ファースト・依存ゼロの日付ピッカー

タイトル（「2026年7月」）をクリックすると年スクロール→月→日の3タップで移動できます。 期間選択はホバーで帯がプレビューされ、端点だけマーカーになります（色は設定可能）。

基本（single）— input に直接タイプもOK（例: 2026/7/5）

期間（range・2ヶ月表示・帯/マーカー色カスタム）

複数選択（multiple）

日時（showTime・5分刻み）

制約（過去とmin/max外・週末が選択不可）

分割フィールド（年select → 月モーダル → 日モーダル連鎖）

フォーム送信（name 指定で hidden 出力 — single / range / multiple）

インライン + マーカー（祝日・予定の表現）
